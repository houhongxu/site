# 主流程

简单来说，就是将入口 index.ts 通过 loader 转换为模块，然后解析为 AST 分析依赖关系构建模块依赖图，再使用各种 plugin 优化模块产物，最后输出文件

Webpack 主流程：

- 初始化：创建 Compiler，应用配置

- 编译： Compiler 创建 Compilation，Compilation 从 entry 入口解析模块，递归构建依赖图，使用 loader 转换文件（JS/CSS/图片等）为 NormalModule 模块

- 优化：Compilation 执行 plugin，代码分割 SplitChunksPlugin、摇树 SideEffectsFlagPlugin、压缩 TerserPlugin 等

- 输出：Compilation 将 NormalModule 模块打包成 chunk ，Compiler 写入 output 目录
