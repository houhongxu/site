主要生命周期钩子类：

- Compiler — 编译器主流程 [源码](https://github.com/webpack/webpack/blob/main/lib/Compiler.js)

- Compilation — 单次编译过程 [源码](https://github.com/webpack/webpack/blob/main/lib/Compiler.js)

- ContextModuleFactory — 上下文模块创建 [源码](https://github.com/webpack/webpack/blob/main/lib/ContextModuleFactory.js)

- MultiCompiler — 多编译器协调 [源码](https://github.com/webpack/webpack/blob/main/lib/MultiCompiler.js)

- JavascriptParser — JavaScript 代码解析 [源码](https://github.com/webpack/webpack/blob/main/lib/JavascriptParser.js)

- RuleSetCompiler - 规则集编译器 [源码](https://github.com/webpack/webpack/blob/main/lib/rules/RuleSetCompiler.js)

## 主要周期

主要周期如下：

initialize → (compile → compilation → make) → (**seal** → **optimize**) → (**processAssets** → emit → done)

未加\*的属于 Compiler
加\*的属于 Compilation
