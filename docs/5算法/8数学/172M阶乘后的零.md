# 166M 分数到小数

## 解题思路

### 思路一

数学题

因为 `2*5=10`，且 2 的个数肯定多于 5，所以只需要计算 5 的个数即可，注意，`3*5`算一个但是`5*5`算两个，所以 25 到 3125 需要额外处理

### 代码

```js
/**
 * @param {number} n
 * @return {number}
 */
var trailingZeroes = function (n) {
  // 因为2*5=10，且2的个数肯定多于5，所以只需要计算5的个数即可，注意，3*5算一个但是5*5算两个，所以25到3125需要额外处理
  return (
    Math.floor(n / 5) +
    Math.floor(n / 25) +
    Math.floor(n / 125) +
    Math.floor(n / 625) +
    Math.floor(n / 3125)
  );
};
```
