# 61M 旋转链表

## 解题思路

### 思路一

快慢指针，注意 k 可能过大的问题，先 k 取余，再用快慢指针找到旋转点即可

### 代码

```js
/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * @param {ListNode} head
 * @param {number} k
 * @return {ListNode}
 */
var rotateRight = function (head, k) {
  if (!head) return null;

  // 计算链表长度将k多余的长度去掉
  let count = 0;
  let temp = head;

  while (temp) {
    temp = temp.next;
    count++;
  }

  k = k % count;

  // 快慢指针找到旋转点
  let fast = head;
  let slow = head;

  while (k-- > 0 && fast.next) {
    fast = fast.next;
  }

  while (fast.next) {
    fast = fast.next;
    slow = slow.next;
  }

  fast.next = head;
  head = slow.next;
  slow.next = null;

  return head;
};
```
