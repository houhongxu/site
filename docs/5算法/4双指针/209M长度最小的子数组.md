# 209M 长度最小的子数组

## 解题思路

### 思路一

双指针滑动窗口

通过 left 和 right 双指针构成滑动窗口，sum 记录窗口和，min 记录结果的最小长度

循环，如果 sum 小就一直加入右边新元素，如果 sum 大于等于就赋值 min，然后去除左边的旧元素

最后返回有效的 min

### 代码

```js
/**
 * @param {number} target
 * @param {number[]} nums
 * @return {number}
 */
var minSubArrayLen = function (target, nums) {
  let left = 0;
  let right = 0;
  let sum = 0;
  let min = nums.length + 1;

  while (right <= nums.length) {
    if (sum < target) {
      sum += nums[right];
      right++;
    } else {
      min = min > right - left ? right - left : min;

      sum -= nums[left];
      left++;
    }
  }

  return min === nums.length + 1 ? 0 : min;
};
```
