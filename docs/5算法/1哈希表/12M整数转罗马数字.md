# 12M 整数转罗马数字

## 解题思路

### 思路一

列出所有情况的 hash，123456789，按从小到大逐位处理，记录每位对应的 0 的个数

### 思路二

只需要特别处理 4,9,40,90,400,900 到数组 hash 里，之后按顺序从大到小一一转为字符串

### 代码

```js
/**
 * @param {number} num
 * @return {string}
 */
var intToRoman = function (num) {
  const val = [1000, 900, 500, 400, 100, 90, 50, 40, 10, 9, 5, 4, 1];
  const rom = [
    "M",
    "CM",
    "D",
    "CD",
    "C",
    "XC",
    "L",
    "XL",
    "X",
    "IX",
    "V",
    "IV",
    "I",
  ];

  let i = 0;
  let res = "";

  while (num) {
    if (num >= val[i]) {
      num -= val[i];

      res += rom[i];
    } else {
      i++;
    }
  }

  return res;
};
```
