# 86M 分隔链表

## 解题思路

### 思路一

首先用虚拟头结点来统一处理头节点

需要两个头节点，large 和 small

然后用三个指针同时遍历链表和大小子链表

大的节点给大链表，小的给小链表

最后根据大链表是否有值来组合

### 代码

```js
/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * @param {ListNode} head
 * @param {number} x
 * @return {ListNode}
 */
var partition = function (head, x) {
  if (!head) return head;

  const largeHead = new ListNode(0, head);
  const smallHead = new ListNode(0, head);
  let small = smallHead;
  let large = largeHead;
  let p = head;

  while (p) {
    if (p.val < x) {
      small.next = p;
      small = small.next;
      p = p.next;
      small.next = null;
    } else {
      large.next = p;
      large = large.next;
      p = p.next;
      large.next = null;
    }
  }

  if (smallHead.next !== largeHead.next) {
    small.next = largeHead.next;
  }

  return smallHead.next;
};
```
