# 151M 反转字符串中的单词

## 解题思路

### 思路一

栈

遍历 s，记录 word

如果是空格且 word 有值就压入栈 stack，否则就记入 word

最后出栈组合成 result

注意 s 可以先最后补一个空格统一遍历格式

### 代码

```js
/**
 * @param {string} s
 * @return {string}
 */
var reverseWords = function (s) {
  const stack = [];
  let word = "";
  let result = "";

  // 统一格式
  s += " ";

  for (let i = 0; i < s.length; i++) {
    const char = s[i];

    if (char === " ") {
      if (word) {
        stack.push(word);
        word = "";
      }
    } else {
      word += char;
    }
  }

  while (stack.length) {
    result += stack.pop();

    if (stack.length) {
      result += " ";
    }
  }

  return result;
};
```
