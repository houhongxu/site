# 55M 跳跃游戏

## 解题思路

### 思路一

按题目处理，使用栈存目录名称，边遍历边记录目录名，碰到/就进行判断是不是.和..来处理

防止最后一轮未处理还要单独处理，path 加一个/即可

### 代码

```js
/**
 * @param {string} path
 * @return {string}
 */
var simplifyPath = function (path) {
  path += "/";
  const stack = [];
  let name = "";

  for (let i = 0; i < path.length; i++) {
    const char = path[i];

    if (char === "/") {
      if (name === "..") {
        stack.pop();
      } else if (name !== "" && name !== ".") {
        stack.push(name);
      }

      name = "";
    } else {
      name += char;
    }
  }

  return "/" + stack.join("/");
};
```
