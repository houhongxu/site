# 139M 单词拆分

## 解题思路

### 思路一

递归，深度优先搜索

首先 wordDict 转 hash

然后 dfs 遍历一个 str，如果 hash[str]通过，就 dfs 剩余字符串

如果 s 空，就说明拼接单词成功

通过 cache[s] 记录搜索过，如果搜过就跳过

注意不是回溯

#### 代码

```js
/**
 * @param {string} s
 * @param {string[]} wordDict
 * @return {boolean}
 */
var wordBreak = function (s, wordDict) {
  const hash = {};
  let result = false;
  const cache = {};

  for (let i = 0; i < wordDict.length; i++) {
    hash[wordDict[i]] = true;
  }

  const dfs = (s) => {
    if (!s) {
      result = true;

      return;
    }

    if (cache[s]) {
      return;
    }

    cache[s] = true;

    for (let i = 0; i < s.length; i++) {
      const str = s.slice(0, i + 1);

      if (hash[str]) {
        dfs(s.slice(i + 1));
      }
    }
  };

  dfs(s);

  return result;
};
```
