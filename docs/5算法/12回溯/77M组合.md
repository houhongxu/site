# 77M 组合

## 解题思路

### 思路一

组合，回溯

用 path 记录路径，通过判断 path 长度跳出递归

通过 arr.slice(i + 1)来避免重复选择 arr[i]

### 代码

```js
/**
 * @param {number} n
 * @param {number} k
 * @return {number[][]}
 */
var combine = function (n, k) {
  const res = [];
  const path = [];
  const nums = [...new Array(n)].map((_, index) => index + 1);

  const dfs = (arr) => {
    if (path.length === k) {
      res.push([...path]);
      return;
    }

    for (let i = 0; i < arr.length; i++) {
      path.push(arr[i]);
      dfs(arr.slice(i + 1));
      path.pop();
    }
  };

  dfs(nums);

  return res;
};
```
