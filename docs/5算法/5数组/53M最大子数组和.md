# 53M 最大子数组和

## 解题思路

### 思路一

子数组最大和，那就是维护一个子数组，遍历时不停对比 max 和 sum

但是子数组不能只 push 不 pop 吧，所以关键在于 sum 小于 0 时，最大和加负数会变小，加正数不如单独一个数大，所以这时直接 pop 到空

### 代码

```js
/**
 * @param {number[]} nums
 * @return {number}
 */
var maxSubArray = function (nums) {
  let max = nums[0];
  let sum = nums[0];
  let arr = [nums[0]];

  for (let i = 1; i < nums.length; i++) {
    // 小于0时，最大和加负数会变小，加正数不如单独一个数大，所以直接重置
    if (sum <= 0) {
      arr = [];
      sum = 0;
    }

    arr.push(nums[i]);
    sum += nums[i];

    if (sum > max) {
      max = sum;
    }
  }

  return max;
};
```
