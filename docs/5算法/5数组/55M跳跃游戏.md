# 55M 跳跃游戏

## 解题思路

### 思路一

贪心，每次尽量跳最远的就行，额外判断一下当值为 0 时失败

### 代码

```js
/**
 * @param {number[]} nums
 * @return {boolean}
 */
var canJump = function (nums) {
  for (let i = 0; i < nums.length; ) {
    if (i + nums[i] >= nums.length - 1) {
      return true;
    }

    if (nums[i] === 0) {
      return false;
    }

    let max = i + nums[i];
    let target = i + 1;

    // 找到当前位置能跳到的最远位置
    for (let j = i + 1; j <= i + nums[i]; j++) {
      if (j + nums[j] > max) {
        max = j + nums[j];
        target = j;
      }
    }

    i = target;
  }

  return false;
};
```
