# 三列均分布局

## 通过 flex

```jsx
<div
  style={{
    height: 200,
    backgroundColor: 'gray',
    display: 'flex',
  }}
>
  <div
    style={{
      flex: 1,
      height: '100%',
      backgroundColor: 'blue',
    }}
  ></div>
  <div
    style={{
      flex: 1,
      height: '100%',
      backgroundColor: 'black',
    }}
  ></div>
  <div
    style={{
      flex: 1,
      height: '100%',
      backgroundColor: 'green',
    }}
  ></div>
</div>
```
